# Proxy
Proxy client using Python.

English:
A simple code snippet that launches a personal proxy client on your machine.
The code is completely open source. 
There's also an additional quickstart file with the settings from the main file.

Русский:
Простой фрагмент кода, запускающий персональный прокси-клиент на вашем компьютере.
Код полностью с открытым исходным кодом.
Также имеется дополнительный файл быстрого запуска с настройками из основного файла.
